# Voom::Presenters

A ruby presentation DSL for describing [Material Design](https://material.io/) user interfaces.

## Demo

    $ git clone git@github.com:rx/presenters.git
    $ cd presenters/
    $ bundle install
    $ rackup

    $ open htt://localhost:9292
